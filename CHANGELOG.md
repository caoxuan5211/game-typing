# 更新日志

## [2.0.0] - 2026-06-23

### 重做
- 将旧版多弹窗打字工具重做为单屏 Code Typing Lab 训练台。
- 新增代码、符号、长段、自定义四类训练模式。
- 新增 WPM、准确率、连击、评级、进度、今日字符目标、连续天数和最近成绩。
- 新增错误键位热区，用于复盘当前训练弱项。

### Bug修复
- 修复 Enter 不能可靠开始游戏的问题：现在 Enter 可开始、暂停后继续、完成后进入下一题。
- 修复主题切换异常：主题统一通过 `data-theme` 和 CSS token 应用。
- 修复 Tab 行为：输入区按 Tab 会插入真实缩进，不会跳出焦点，也不会作为普通可见字符显示。

### 体验改进
- 取消完成弹窗阻断，成绩和建议直接在训练台更新。
- 增加 reduced-motion 支持和更清晰的键盘焦点样式。
- 更新 README、version.json 和产品上下文文档。

## [1.0.0] - 2026-06-23

### ✨ 新增功能
- **模块化架构**: 完全重构为ES6模块化设计，清晰的关注点分离
- **5种主题**: 紫罗兰、海洋蓝、日落橙、森林绿、午夜黑
- **成就系统**: 5个可解锁成就（速度恶魔、完美主义者、马拉松选手、稳定发挥、夜猫子）
- **自定义文本模式**: 用户可以输入自己的练习文本
- **音效系统**: 使用Web Audio API实现实时按键反馈
- **统计面板**: 详细的游戏统计和高分榜
- **本地存储**: 成绩、设置和成就的持久化存储

### 🎨 UI/UX 改进
- 现代化设计，流畅的动画效果
- 响应式布局，支持所有屏幕尺寸
- 进度条带颜色过渡
- 模态框系统（主题、统计、自定义文本）
- 成就解锁通知
- Toast消息提示
