# Product

## Register

product

## Users
Programmers and programming learners who want short, focused code typing drills in a browser. They use it during practice breaks or deliberate training sessions to improve symbol accuracy, indentation fluency, and code entry rhythm.

## Product Purpose
Code Typing is a training game for code-oriented typing. Success means a player can start quickly from the keyboard, see precise live feedback, finish repeatable drills, and feel motivated to improve through clear progress, streaks, records, and practical practice modes.

## Brand Personality
Focused, sharp, encouraging. The product should feel like a compact training cockpit rather than a marketing page or toy.

## Anti-references
Avoid decorative landing-page layouts, overdone gradients, modal-heavy flows, unreadable low-contrast themes, and interactions that fight coding muscle memory such as broken Enter start behavior or Tab leaving the typing field.

## Design Principles
- Keyboard first: every common play action should work naturally without reaching for the mouse.
- Train code reality: indentation, symbols, punctuation, and multiline snippets are first-class practice material.
- Feedback in flow: show progress, mistakes, rhythm, and next goals without interrupting the drill.
- Motivation without clutter: use records, streaks, quests, and grades to encourage another run.
- Robust before flashy: state transitions, theme switching, input handling, and persistence must be predictable.

## Accessibility & Inclusion
Target WCAG AA contrast, visible focus states, reduced-motion support, keyboard operability, and color-independent error feedback.
