### 🎵 音效系统

**7种场景音效**
1. `playKeystroke()` - 正确按键的清脆点击
2. `playError()` - 错误的低沉警告
3. `playStart()` - 上升和弦 (C-E-G)
4. `playComplete()` - 胜利号角 (4音符)
5. `playComboMilestone()` - 连击闪耀音效
6. `playTone()` - 通用音调
7. `playAmbient()` - 保留的环境音

**技术特性**
- Web Audio API
- ADSR 音量包络
- 主音量控制
- 多音轨并发
- 优雅降级

### 🎯 使用指南

**开始训练**
1. 访问 https://type.mineguai.com/train.html
2. 选择训练模式 (代码/符号/长段/自定义)
3. 选择难度 (简单/标准/困难)
4. 点击"开始训练"或按 Enter
5. 直接在代码区开始打字
6. 按 ESC 暂停，Enter 继续

**查看统计**
1. 访问 https://type.mineguai.com/stats.html
2. 查看最佳成绩、训练次数、连续天数
3. 查看今日训练进度
4. 浏览详细的历史记录
