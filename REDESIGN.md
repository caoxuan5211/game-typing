# Code Typing Lab v3.0.0 - 优雅重设计

## 🎨 设计理念

这次重设计遵循优雅、克制的设计原则，采用古典杂志风格的排版，打造专业而舒适的打字训练体验。

### 视觉设计

**色彩方案**
- 背景：柔和米白色 (#F7F5F1)
- 表面：纯白 (#FFFFFF)
- 边框：淡雅米色 (#E7E3DA)
- 文字：深灰墨色 (#1E2227)
- 强调色：克制的石板蓝 (#3C5A78)

**字体系统**
- 标题：Playfair Display (高对比度衬线体，优雅而克制)
- 正文：Inter (现代无衬线体，清晰易读)
- 代码：SF Mono / Consolas (等宽字体，专业编程体验)

**布局特点**
- 单列居中布局，留白充足
- 经典的卡片式设计，柔和的圆角和阴影
- 响应式网格系统，适配各种屏幕尺寸
- 清晰的视觉层次和信息架构

### 交互设计

**音效系统**
- ✨ 全新的音频引擎 (audio.js)
- 🎹 按键反馈：清脆的点击音
- ❌ 错误提示：低沉的警告音
- 🎯 开始训练：上升和弦
- 🎊 完成训练：胜利号角
- 🔥 连击里程碑：闪耀音效 (10/25/50/100连击)
- 🎚️ 主音量控制和开关

**视觉反馈**
- 实时字符高亮 (正确/错误/当前)
- 进度条平滑过渡动画
- 状态指示器脉动效果
- 按钮悬停和点击状态

## 🎯 技术亮点

### 音频系统特性
- **AudioContext 单例管理**：避免资源泄漏
- **增益节点控制**：主音量统一管理
- **ADSR 包络**：平滑的音量包络线
- **多音轨支持**：可同时播放多个音效
- **降级处理**：音频不可用时静默失败

### CSS 变量系统
```css
:root {
    --serif: 'Playfair Display', Georgia, serif;
    --sans: 'Inter', system-ui, sans-serif;
    --mono: 'SF Mono', 'Consolas', monospace;
    
    --spacing-unit: 8px;
    --radius: 12px;
    --shadow: 0 4px 12px rgb(0 0 0 / 0.06);
}
```

### 响应式断点
- Desktop: > 1024px
- Tablet: 768px - 1024px
- Mobile: < 768px
- Small Mobile: < 480px

## 🎨 设计决策

### 为什么选择 Playfair Display？
- 高对比度衬线字体，典雅而专业
- 适合大尺寸标题使用
- 与 Inter 正文字体形成完美搭配
- 传达出"专业工具"的质感

### 为什么选择石板蓝作为强调色？
- 克制而专业，不喧宾夺主
- 与米白背景形成柔和对比
- 符合代码编辑器的视觉习惯
- 避免使用过于鲜艳的颜色（紫色/橙色）

### 为什么采用卡片式布局？
- 清晰的信息分组
- 适合桌面和移动端
- 易于扫描和理解
- 现代而不过时
