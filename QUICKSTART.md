# 快速启动指南

## 🚀 5分钟上手

### 1. 克隆项目

```bash
git clone https://github.com/caoxuan5211/game-typing.git
cd game-typing
```

### 2. 启动本地服务器

**方式一：使用 Python**
```bash
python -m http.server 8000
```

**方式二：使用 Node.js**
```bash
npx http-server -p 8000
```

**方式三：使用 VS Code**
- 安装 "Live Server" 扩展
- 右键 index.html，选择 "Open with Live Server"

### 3. 访问应用

打开浏览器访问：`http://localhost:8000`

---

## 🎮 基本使用

### 开始游戏
1. 选择练习模式（代码片段/符号/混合/自定义）
2. 选择难度（简单/中等/困难）
3. 点击"开始练习"按钮
4. 根据显示的文本进行输入
5. 完成后查看详细统计

### 快捷键
- `Ctrl/Cmd + Enter` - 开始练习
- `ESC` - 暂停/关闭模态框

### 功能介绍
- **主题切换**: 点击右上角太阳图标
- **音效开关**: 右上角的开关按钮
- **查看统计**: 点击统计图表图标
- **自定义文本**: 选择自定义模式，输入你的文本

---

## 📊 统计指标说明

- **WPM (Words Per Minute)**: 每分钟单词数，标准为5个字符=1个单词
- **准确率**: 正确字符占总输入字符的百分比
- **CPM (Characters Per Minute)**: 每分钟字符数
- **稳定性**: 考虑删除修正的击键准确性
- **修正次数**: 使用退格键删除的次数

## 🔧 故障排除

### 音效不工作
- 确保浏览器允许音频自动播放
- 检查音效开关是否打开
- 尝试刷新页面

### 统计不保存
- 检查浏览器是否禁用了LocalStorage
- 清除浏览器缓存后重试
- 确保不在隐私/无痕模式

### 显示异常
- 刷新页面（Ctrl/Cmd + R）
- 清除浏览器缓存
- 尝试其他浏览器

### 输入延迟
- 关闭浏览器扩展
- 关闭其他标签页
- 检查电脑性能

## 🚀 部署到服务器

### 使用部署脚本

```bash
# 编辑 deploy.sh 配置你的服务器信息
vim deploy.sh

# 运行部署
./deploy.sh
```

### 手动部署

```bash
# 连接到服务器
ssh your-server

# 克隆或更新代码
cd /var/www/html
git clone https://github.com/caoxuan5211/game-typing.git
# 或
cd game-typing && git pull

# 设置权限
chmod -R 755 game-typing
```

## 💡 提示和技巧

### 提高速度
1. 保持正确的手指位置
2. 不要看键盘
3. 专注于准确性而不是速度
4. 定期练习

### 提高准确率
1. 慢一点，但要准确
2. 集中注意力
3. 熟悉常用符号的位置
4. 练习特殊字符

### 最佳实践
1. 每天练习15-30分钟
2. 从简单模式开始
3. 逐步增加难度
4. 记录你的进步
