# Code Typing Lab

> 为程序员设计的优雅代码打字训练平台

## ✨ 特性

- 🎨 **优雅设计** - 古典衬线字体 + 克制的配色
- 🎵 **丰富音效** - 7种场景音效，实时反馈
- 📊 **数据统计** - WPM、准确率、连击、每日目标
- 🎯 **多种模式** - 代码片段、符号密集、长段落、自定义
- 🌓 **主题切换** - 浅色/深色主题支持
- 📱 **响应式** - 完美适配桌面和移动设备
- 💾 **本地存储** - 自动保存训练数据

## 🚀 在线体验

**https://type.mineguai.com**

## 🎨 v3.0.0 重设计亮点

这个版本带来了完整的视觉和交互重设计：

- **字体系统**：Playfair Display 衬线标题 + Inter 正文 + SF Mono 代码
- **配色方案**：柔和米白背景 + 石板蓝强调色，专业而优雅
- **音效系统**：全新的 Web Audio API 引擎，丰富的声音反馈
- **交互优化**：平滑动画、实时高亮、连击里程碑

详见：[REDESIGN.md](REDESIGN.md)

## 🛠️ 本地开发

```bash
# 克隆仓库
git clone https://github.com/caoxuan5211/game-typing.git
cd game-typing

# 启动本地服务器
python -m http.server 8080
# 或者
npm run dev

# 访问 http://localhost:8080
```

## 📦 技术栈

- 纯 JavaScript (ES6 模块)
- Web Audio API (音效系统)
- CSS 变量 (主题系统)
- LocalStorage (数据持久化)

## 🎯 训练模式

- **代码模式** - 真实代码片段，包含 Tab 缩进
- **符号模式** - 密集符号训练，提升特殊字符速度
- **长段模式** - 完整函数和逻辑块，训练持续专注力
- **自定义** - 粘贴你自己的代码进行训练
