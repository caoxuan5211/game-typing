### ⭐ 等级系统 (10个等级)

| 等级 | 所需XP | 称号 |
|------|--------|------|
| Lv.1 | 0 | 新手 |
| Lv.2 | 100 | 学徒 |
| Lv.3 | 250 | 熟练 |
| Lv.4 | 500 | 精通 |
| Lv.5 | 1000 | 专家 |
| Lv.6 | 1800 | 大师 |
| Lv.7 | 3000 | 宗师 |
| Lv.8 | 5000 | 传奇 |
| Lv.9 | 8000 | 神话 |
| Lv.10 | 12000 | 至尊 |

**经验值计算公式:**
```javascript
baseXP = 字符数 / 5
wpmBonus = (WPM / 10) * 5
accuracyBonus = 准确率 >= 95% ? 20 : 准确率 >= 90% ? 10 : 0
totalXP = baseXP + wpmBonus + accuracyBonus
```

**示例:**
- 输入 100 个字符，60 WPM，96% 准确率
- XP = 20 + 30 + 20 = **70 XP**
